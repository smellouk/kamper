| Boundary | Description |
|----------|-------------|
| Git release tag → CI workflow `if` filter | release-please creates tags; workflow filters route them to specific publish jobs. Mismatch silently skips publication. |
| CocoaPods → Gradle | Podspec scripts invoke `:libs:xcframework:assemble...` Gradle tasks during `pod install`. Wrong path stalls iOS builds. |
| npm → filesystem | `file:../../libs/ui/rn` resolves at `npm install` time. Wrong path causes RN demo build failure. |
