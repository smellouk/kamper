- **Found during:** Task 2 verification
- **Issue:** The BSD sed word-boundary pattern `[[:<:]]Kamper[[:>:]]` did not match `KamperColors` because it is followed by uppercase `C` and macOS sed word boundaries are whitespace-adjacent. The compound names like `KamperColors` are neither in the explicit substitution list nor caught by the word-boundary rule.
- **Fix:** Manual `sed -i '' 's/KamperColors/KonitorColors/g; s/LocalKamperColors/LocalKonitorColors/g'` on KonitorTheme.kt
- **Files modified:** `libs/ui/kmm/src/commonMain/kotlin/com/smellouk/konitor/ui/compose/KonitorTheme.kt`
